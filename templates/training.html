Common error you might see in training:

1. The bitmap you uploaded might not have a match text file, make sure text file is there

2. You might put additional options in the Additional Options field, which is not supported by tesstrain Solution: Check all option in https://github.com/tesseract-ocr/tesstrain/blob/main/tesstrain-main.sh

3. It is good that you only have one letter per image instead put all letters in one image for training

4. If you provide a start Model name such as "eng", the result file will be more than 1G in size. To save space, every 2AM, the .traindata files will be deleted if it is more than 24 hours old

5. If the log is not refresh, try to go log to click to see possible error message

6. There is only one training process running at a time, if you want to train another model, you need to wait for the current training to finish
